Error handling. Default is ErrorMode.Default.

• Optional keepPreviousState: boolean

Keep previous state until new request resolves rather than clearing the state as undefined. Default is false.

PnpHookOptions.keepPreviousState

list​

• Optional list: string

List GUID Id or title for getting list field. Keep undefined for web field. Changing list value repeats request.
